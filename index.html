How to add your member card

New to GitHub? Use the GitHub website path. Already comfortable with Git? You can clone the repo and work locally instead.

This wall is PUBLIC. Only add profile details and images you are comfortable sharing online.

New to Git and GitHub? Joey Salatino made a helper doc: open the guide.

Option 1: Edit on GitHub.com

  1. Step 1: Make your profile Make a GitHub account and send your username in github-username-drop. Then open members/example-member.json, copy its contents, and create a new file in members/ named after yourself.
  2. Step 2: Fill out your info Edit your new JSON file with your name, major, interests, GitHub username, optional image, and anything else you are comfortable sharing publicly.
  3. Step 3: Add yourself to the list Edit members.json and add your new file path. This registers your profile file so the website knows to display it.
  4. Step 4: Open a pull request When GitHub asks how to save your changes, choose to create a new branch and open a pull request. Someone will review it before it goes live.
  5. Step 5: See yourself live After your pull request is approved and merged, GitHub Pages updates and your card appears below. Scroll down to see an example member card.

Option 2: Clone the repo locally

  1. Step 1: Clone the repository Clone the repo to your computer and open it in your code editor.
  2. Step 2: Create a new branch Make a branch for your profile so your changes stay separate from the main website.
  3. Step 3: Make your profile file Copy members/example-member.json into a new file in members/ and fill it out with your information.
  4. Step 4: Add yourself to the list Edit members.json and add your new file path. This registers your profile file so the website knows to display it.
  5. Step 5: Commit, push, and open a pull request Commit your changes, push your branch to GitHub, and open a pull request for review.
