Add yourself in five small steps

This wall is public. Only add profile details and images you are comfortable sharing online.

  1. Start from the example Copy members/example-member.json into /members and rename it with your name.
  2. Fill in your profile Add your name, class year, major, interests, short bio, and optional profile image path.
  3. List your file Add the new JSON path to members.json so the page knows to load it.
  4. Preview locally Run a small local server and confirm your card, tags, bio, and image all look right.
  5. Open the pull request Commit your JSON and image, open a PR, and your card appears here after it is merged.
