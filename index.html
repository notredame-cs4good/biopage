Add yourself with GitHub

This wall is public. Only add profile details and images you are comfortable sharing online.

New to Git and GitHub? Joey Salatino made a helper doc: open the guide.

  1. Get access Make a GitHub account, then send your username in github-username-drop so we can add you.
  2. Make a branch After you have access, clone the repo and run git checkout -b add-your-name.
  3. Create your file Copy members/example-member.json to a new file like members/jane-doe.json.
  4. Edit your profile Update your JSON with your name, year, major, interests, bio, and optional image in assets/.
  5. Update the list Open members.json and add your new file path so the site knows your card exists.
  6. Send it for review Commit, push your branch, and open a pull request. Your card appears after the PR is approved and merged.
